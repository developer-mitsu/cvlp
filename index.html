ABOUT

Code Villageとは

プログラミング教室Code Villageは未経験から多様な働き方の選択肢を持つ事を目的としたスパルタ教室です。

私達はプログラミングスキルは誰しもが独学で習得出来る技術と考えており、スクールの存在意義は豊富な教材と優秀な講師、学習のマイルストーンによる技術習得の時短だと捉えております。

Code Villageでは独学で習得する自信や、モチベーションのある方こそ利用して頂きたい仕組みとなっており、レベルに上限のないカリキュラム、集団から個別への飛び級制度などを導入する事によって頑張る人を最大限応援するスクールとなっております。

1日3H以上の自学習強制による学習の習慣化、そしてチーム開発やペアプロなど実務を意識したカリキュラムで短期間で圧倒的な成長を目指します。

学習プログラムの特徴

1コミュニティ

Code Villageには志・学習意欲共に高い生徒が多数在籍しており、校外の時間でクラスを跨いだ交流が行われております。中には共に起業したり、プロダクトを共同開発している生徒様もいらっしゃいます。

2詰め込み授業

Code Villageでは長期間ダラダラ学習するのではなく、短期間で圧倒的な学習量を求め、約300時間以上 (最低)の課題の取り組みと、18回の直接講義の参加を強制化しております。学習する習慣と、技術の基礎を習得する事により卒業後の自走を後押ししております。

3習熟度別授業

基本的に未経験者(初学者)を対象としておりますが、数週間で生徒のレベルが二極化して参ります。 講義の中間でクラス分けを行う事により、双方有意義な受講をする事が可能です。進捗上限はなく、更に向上心と結果の伴う生徒様には最大個別コース移行も行なっております。

FLOW

講義フロー

1事前課題

入校前に事前課題としてhtml/cssの基礎理解をして頂きます。
簡易的な1枚のHPのコーディングが出来るまでの学習のマイルストーンを提供し、受講前に基礎理解をして頂いた上で更なる技術向上を目指します。

2対面講義

全18回の対面による集団講義を受講して頂きます。 強制化された自学習に加え、事前課題による基礎知識を習得した上での講義実施になりますので、 より一層レベルの高いコンテンツ提供が可能になります。

3自主学習

受講期間中は1日3H以上の自学習を強制しております。 量を伴う復習/予習を行う事により、学習の習慣化と短期的な成長を見込む事が可能となります。

4コミュニティを活かす

過去の生徒様の中には共に起業を試みたり、共に受託開発を行う生徒も多数存在致します。今後も成長するコミュニティをご自身の技術習得、キャリア形成に活かしてください。 また、弊社が不定期に開催するイベントに参加して頂く事も可能となっております。卒業して終わりの関係値ではなく当社、生徒様共に成長出来る関係形成を目指しております。

CURRICULUM

カリキュラム

受講期間の3ヶ月で、ゼロからモダンな企業の現場で通用するフロントエンドエンジニアを育成します。
本スクールの育成方針は、ひとえに「現場で必要とされるエンジニア」です。
その絶対命題のもと、以下の指針に基づいてカリキュラムを設計しています。

フロントエンド初級

HTML/CSSの基礎を学習し、基礎的なマークアップ技術を身に着けます。 課題に取り組み、受講期間中を通してマークアップの経験値を積むことで、高い水準まで技術を引き上げます。

  • HTML
  • CSS
  • jQuery

フロントエンド中級

複数デバイスへの対応や、アニメーション、設計等の一歩踏み込んだCSSの技術を習得します。また、javascriptの基礎文法・DOMの概念を学習します。

  • レスポンシブコーディング
  • アニメーション
  • javascript
  • CSS設計ノウハウ

フロントエンド上級

トレンドに則った技術を学習し、より実践的な開発を行います。また、Git等を用いたチーム開発技法を学習します。

  • Yarn・webpack
  • フロントエンドフレームワーク
  • Git/Github
