CURRICURUM

カリキュラム

受講期間の3ヶ月で、ゼロからモダンな企業の現場で通用するフロントエンドエンジニアを育成します。
本スクールの育成方針は、ひとえに「現場で必要とされるエンジニア」です。
その絶対命題のもと、以下の指針に基づいてカリキュラムを設計しています。

フロントエンド初級

学習期間1ヶ月

HTML/CSSの基礎を学習し、基礎的なマークアップ技術を身に着けます。 課題に取り組み、受講期間中を通してマークアップの経験値を積むことで、高い水準まで技術を引き上げます。

HTML

Webページを作成する為の言語です。
ページの構造を定義したり、リンクを貼ったりすることが出来ます。

CSS

HTMLで構造を定義したWebページに対して、装飾を施すための言語です。

jQuery

jQueryを用いると、比較的容易にwebページに動きをつけることが出来ます。
ただし、この領域に関してはそれほどの時間はかけません。
とはいえ、便利なプラグインなども多数存在しているので、文法や各プラグインのドキュメントの読み方・実装方法など、重要な部分に絞って学習します。

フロントエンド中級

学習期間1ヶ月

レスポンシブコーディング

タブレットやスマホ等、ユーザーはPCだけではなく、様々なデバイスでwebページを閲覧します。各デバイスに対応したコーディング(レスポンシブコーディング)を行うための、各種ノウハウを習得します。

アニメーション

主にCSSを用いたアニメーションを学習します。
ボタンやメニューなど、webページ上の要素に様々なエフェクトをつける為の技術を習得します。

JavaScript

Webページで使用されるプログラミング言語です。 ポップアップウィンドウなどのブラウザの機能を制御や、
スライドショーやスクロールによる挙動の変化など、様々な実装が可能になります。基礎文法からDOMの扱い方まで、丁寧に解説します。
また、イベントバブリング・クロージャ・thisの挙動の違い・変数の巻き上げなど、初学者が引っかかりがちな部分に関しては、特に注意しながら解説します。

フロントエンド上級

学習期間1ヶ月

HTML/CSSの基礎を学習し、基礎的なマークアップ技術を身に着けます。 課題に取り組み、受講期間中を通してマークアップの経験値を積むことで、高い水準まで技術を引き上げます。

Git

現場で最も使用されるバージョン管理システムです。 表面だけ学習し、わかったつもりになることは比較的容易にできます。 しかし、プロジェクト単位で使用されるシステムである性質上、上辺だけの理解では一つのミスでチーム全体に多大な損害を与える可能性があります。 そのため、誤った理解をすることの無いよう丁寧にgitを学習していきます。

  • git基礎
  • mergeコンフリクトへの対応
  • 実践的なコミットメッセージの記法
  • ブランチ管理手法(git flow, github flow)
  • etc…
